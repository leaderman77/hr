## Cradle HR system
- [minicondani](https://docs.conda.io/en/latest/miniconda.html) o'rnating
- Repo boshidan (root) quyidagilarni qiling:
    ```shell
    conda install -c conda-forge mamba
    mamba env create -f environment.yml
    conda activate cradle_hr
    pip install -e .
    pre-commit install
    ```
